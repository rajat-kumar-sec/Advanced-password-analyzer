## Quick Start.

### Local Development.

```bash
pip install -r requirements.txt
python app.py
```

Then open `http://localhost:5000`

### Docker

```bash
docker compose up
```

### Deploy to Render

1. Push to GitHub
2. Go to https://render.com
3. Connect GitHub repo
4. Create Web Service
5. Set Start Command: `gunicorn app:app`
6. Deploy!
